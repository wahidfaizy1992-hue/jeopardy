JEOPARDY! GAME RULES

Welcome to Jeopardy Game Studio!

Three players compete by answering clues from six categories. The player with the highest score at the end of the game wins.

Before the Game Begins

  1. Enter the names of the three players.
  2. Select six different categories.
  3. Click Start Game.

Each category contains five clues worth:

$100
$200
$300
$400
$500

*Note: Higher-value clues are generally more difficult.

How to Play

  1. Player 1 begins the game by selecting any clue from any category.
  2. The selected clue is displayed, and the timer starts.
  3. The current player must answer the clue before the timer expires.
  4. The host marks the answer as Correct or Incorrect.
  5. The player's score is updated automatically.
  6. The clue is removed from the board and cannot be selected again.
  7. The turn passes to the next player.

Turn Order

The turns continue in this order until all clues have been played:

Player 1
→
Player 2
→
Player 3
→
Repeat

Scoring

Correct Answer

Adds the clue's value to the player's score.

Example: +$300 points

Incorrect Answer

Subtracts the clue's value from the player's score.

Example: −$400 points

Timer & Played Clues

⏱️ The Countdown

Each clue has a countdown timer. Players must answer before time expires. If the timer reaches zero before an answer is given, the turn ends immediately.

🔒 Spent Clues

After a clue has been played, it disappears from the board and cannot be selected again for the remainder of the match.

Winning the Game

  • The game ends when all 30 clues have been played.
  • The player with the highest total score is declared the winner!
  • If two or more players finish with the same score, the game ends in a tie unless the host chooses to play a tie-breaker question.

Good Sportsmanship

  • Wait for your turn.
  • Do not interrupt another player's answer.
  • Respect the host's final decisions.
  • Enjoy the game and have fun!
